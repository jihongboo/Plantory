# 一、产品概述

## 1.1 产品名称

Plantory

副标题：

```
AI Plant Doctor
```

---

## 1.2 产品定位

一个面向 **新手养花用户** 的 AI 植物助手。

核心能力：

```
拍照 → AI诊断植物问题 → 提供养护建议
```

同时提供：

* 植物管理
* 基础养护指南
* 虚拟植物状态

---

## 1.3 目标用户

主要用户：

```
新手养花用户
```

典型特征：

* 不懂植物
* 家里有 1–5 盆植物
* 经常遇到植物问题

例如：

```
叶子发黄
植物枯萎
叶子有斑点
```

---

## 1.4 核心价值

解决用户一个核心问题：

```
植物出问题
↓
不知道原因
↓
不知道怎么办
```

Plantory：

```
拍照
↓
AI诊断
↓
给出解决方案
```

---

# 二、产品目标

MVP目标：

```
验证用户是否愿意使用 AI 植物诊断
```

关键指标：

```
AI诊断次数
用户留存
付费转化
```

成功标准：

```
付费率 > 5%
```

---

# 三、功能范围（MVP）

MVP只包含 **5个核心模块**：

```
植物管理
AI植物诊断
植物养护信息
虚拟植物状态
订阅付费
```

不做：

```
社区
复杂植物数据库
电商
社交
```

# 四、功能设计

# 4.1 植物管理

用户可以管理自己的植物。

### 数据结构

```
Plant
```

字段：

```
id
name
species
photo
created_at
note
health_status
```

health_status：

```
healthy
warning
critical
```

---

### 功能

用户可以：

```
添加植物
编辑植物
删除植物
查看植物
```

---

### 添加植物流程

```
点击添加植物
↓
拍照或上传照片
↓
输入名称
↓
选择植物类型（可选）
↓
保存
```

---

### UI结构

```
我的植物

+ 添加植物

植物列表
- 龟背竹
- 多肉
- 绿萝
```

---

# 4.2 AI植物诊断（核心功能）

用户可以拍照让 AI 诊断植物问题。

---

### 用户流程

```
点击诊断植物
↓
拍照
↓
上传
↓
AI分析
↓
返回结果
```

---

### AI返回内容

```
植物种类
问题类型
可能原因
解决建议
```

---

### 示例结果

```
植物：龟背竹

问题：
叶子发黄

可能原因：
1. 浇水过多
2. 光照不足

建议：
减少浇水频率
放到明亮散射光环境
```

---

### 数据结构

```
Diagnosis
```

字段：

```
id
plant_id
photo
result
created_at
```

---

# 4.3 植物养护指南

当识别植物后展示基础信息。

MVP仅支持：

```
20-30 种常见植物
```

例如：

```
龟背竹
绿萝
多肉
虎皮兰
发财树
```

---

### 数据字段

```
PlantGuide
```

字段：

```
species
light
water
temperature
fertilizer
```

---

### 示例

```
龟背竹

光照：
明亮散射光

浇水：
7-10天一次

温度：
18-30℃
```

---

# 4.4 虚拟植物状态

每个真实植物都有一个 **虚拟版本**。

状态与 AI 诊断结果关联。

---

### 状态类型

```
healthy
warning
critical
```

---

### UI表现

使用不同图片：

```
healthy.png
warning.png
dying.png
```

---

### 状态变化规则

```
AI判断健康 → healthy
AI检测问题 → warning
AI检测严重问题 → critical
```

# 五、AI技术方案

MVP采用 **通用视觉模型 + LLM**。

架构：

```
App
↓
Backend
↓
AI API
```

---

## AI平台

推荐：

OpenAI

模型：

```
GPT-4o mini (vision)
```

能力：

```
识别植物
分析问题
生成建议
```

---

### Prompt示例

```
You are a professional plant expert.

Analyze the plant in this photo.

Return:

1 plant species
2 possible problems
3 causes
4 solutions

Limit answer to 150 words.
```
